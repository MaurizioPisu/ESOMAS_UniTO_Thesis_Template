# ESOMAS_UniTO_Thesis_Template
This unofficial LaTeX thesis template is designed to make the thesis writing process easier and faster. It follows the general formatting and structural requirements commonly requested by professors at the ESOMAS Department (University of Turin).

⚠️ **Note:** This template is based on common students’ experience. It may not fully match the specific requirements of every professor. Please verify the guidelines provided by your supervisor or department.

---

## Repository Structure

The repository contains three main components:

- **`PDFexample.pdf`** – A compiled version of the thesis, showing the final result.
- **`ThesisTemplate/`** – The LaTeX source code.
- **`ZipThesisTemplate.zip`** – A ready-to-upload ZIP file for **Overleaf**.  
  To use it:
  1. Go to *Overleaf -> New project -> Upload project* 
  2. Upload `ZipThesisTemplate.zip`
  3. Start editing your thesis online.
